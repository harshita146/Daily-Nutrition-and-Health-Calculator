# Nutrition & Health Report Script

## Overview

This Python script collects user inputs including name, age, weight, height, gender, and physical activity level. It then calculates personalized nutrition and health recommendations based on the input data.

## Functionalities

- Calculates daily protein, carbohydrate, fat, and calorie requirements based on activity level and weight.
- Computes Body Mass Index (BMI) and categorizes weight status.
- Determines ideal weight and weight difference compared to current weight.
- Estimates daily water intake based on weight.
- Suggests recommended sleep duration based on age.
- Provides vitamin and mineral intake recommendations based on gender.

## Inputs

- Name (string)
- Age (integer)
- Weight in kilograms (float)
- Height in meters (float)
- Gender (M/F/O)
- Activity level (sedentary/moderate/active/athlete)

## Outputs

- BMI value and category (underweight, normal, overweight, obesity)
- Ideal weight and weight difference
- Daily nutritional requirements (protein, carbohydrates, fats, calories)
- Daily water intake in liters
- Recommended daily sleep hours
- Recommended vitamins and minerals intake

## Usage

Run the script, enter the prompted values, and receive a detailed daily nutrition and health report personalized to the individual's information.
