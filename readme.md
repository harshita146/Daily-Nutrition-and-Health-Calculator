# Daily Nutrition & Health Calculator

A Python console application that provides personalized daily nutrition and health recommendations based on user input. Get instant reports on BMI, daily nutrient needs, vitamins, water, and sleep requirements.

## Features

- Calculates BMI and categorizes health status (Underweight, Normal, Overweight, Obesity)
- Estimates ideal body weight and difference from actual weight
- Provides protein, carbohydrate, fat, and calorie requirements based on activity level
- Suggests daily vitamins and minerals according to gender
- Calculates water intake and recommended sleep duration
